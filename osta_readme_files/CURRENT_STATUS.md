Two of its rules are worth knowing before touching any endpoint: `/auth/refresh` takes the refresh token as a `Bearer` header with an empty body, and multipart bodies only parse on `POST` (use `_method=PUT`). Both are written up in [guides/09_api_endpoints.md](guides/09_api_endpoints.md).
